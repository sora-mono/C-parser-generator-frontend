条件控制块基类（if、for、while、switch等） 

1.控制块主体例： if(condition) \{ // 主体 \} else \{ // 主体 \} for(int i = 0;i $<$ x;i++)\{ // 主体 \} 2.控制块条件指判断控制块是否执行/跳转方向的语句，如1中if语句的condition， for中的i $<$ x; \begin{DoxyAttention}{注意}
