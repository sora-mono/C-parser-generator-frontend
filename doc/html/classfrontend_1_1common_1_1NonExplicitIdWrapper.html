不强制显示调用构造函数使用默认无效值版

参数
[in]IdType_： 要包装的ID的实际类型
[in]LabelEnum_：分发标签的枚举类型，与label_共同决定唯一的ID
[in]label_：分发标签枚举类型中的具体枚举，与LabelEnum_共同决定唯一的ID类型 例： enum class ExampleEnum { kExampleType }; using Id = NonExplicitIdWrapper<size_t,ExampleEnum, ExampleEnum::kExampleType>;
