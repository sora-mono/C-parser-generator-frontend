template<class IdType_, class LabelEnum_, const LabelEnum_ label_, const IdType_ invalid_value_>
class frontend::common::BaseIdWrapper< IdType_, LabelEnum_, label_, invalid_value_ >

注意
基础ID包装器类，不应直接使用
参数
[in]IdType_： 要包装的ID的实际类型
[in]LabelEnum_：分发标签的枚举类型，与label_共同决定唯一的ID
[in]label_：分发标签枚举类型中的具体枚举，与LabelEnum_共同决定唯一的ID类型
[in]invalid_value_：第四个参数为无效值

使用例： enum class ExampleEnum { kExampleType }; using Id = BaseIdWrapper<size_t,ExampleEnum,ExampleEnum::kExampleType, static_cast<size_t>(-1)>;
