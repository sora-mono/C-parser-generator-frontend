template<class IdType_, class LabelEnum_, const LabelEnum_ label_>
class frontend::common::ExplicitIdWrapper< IdType_, LabelEnum_, label_ >

强制显式调用构造函数且使用默认无效值版Wrapper

参数
[in]IdType_： 要包装的ID的实际类型
[in]LabelEnum_：分发标签的枚举类型，与label_共同决定唯一的ID
[in]label_：分发标签枚举类型中的具体枚举，与LabelEnum_共同决定唯一的ID类型

使用例： enum class ExampleEnum { kExampleType }; using Id = ExplicitIdWrapper<size_t,ExampleEnum,ExampleEnum::kExampleType>;
