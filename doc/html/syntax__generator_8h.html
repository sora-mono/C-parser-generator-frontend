语法分析机配置生成器

1.语法分析机配置生成器使用LALR(1)语法 2.支持非终结产生式空规约 3.支持运算符优先级（二义性文法） 4.支持双目和左侧单目语义共存 5.支持产生式循环引用 6.向前看符号不是运算符时采用移入优先原则 例： IF -> if() {} else if() {} 优先选择if-else路径
