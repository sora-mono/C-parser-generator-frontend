根据给定产生式体和下一个移入的产生式位置获取下一个移入的产生式ID

参数
[in]production_body_id：子产生式体标号
[in]next_word_to_shift_index：产生式体中产生式ID的位置
返回
返回production_body_id和next_word_to_shift_index指定的产生式ID

point_index越界（大于等于production_body_id指定的子产生式中产生式数目）时 返回ProducNodeId::InvalidId() production_body_id不得超出该产生式拥有的产生式体个数 终结产生式和运算符产生式只存在一个子产生式，子产生式中只含有一个产生式， 所以应使用 ProductionBodyId(0),NextWordToShiftIndex(0)才可以获取有效产生式ID

注解
使用举例： 产生式例： IdOrEquivence -> ConstTag Id IdOrEquivence -> IdOrEquivence "[" Num "]" IdOrEquivence -> IdOrEquivence "[" "]" IdOrEquivence -> ConstTag "*" IdOrEquivence 获取IdOrEquivence产生式内第二个子产生式中第三个产生式（"Num"）的ID
NonTerminalProductionNode example_node;
example_node.GetProductionNodeInBody(ProductionBodyId(1),
NextWordToShiftIndex(2));
注意
所有派生类均应重写该函数，不得改变原始语义
